# etch-a-sketch

The Odin Project - Etch a Sketch

Etch a Sketch is a browser version of something between a sketchpad and an Etch-A-Sketch.

Lesson Application
* HTML5
* CSS 
* JavaScript
* DOM Manipulation
* Git Workflow
